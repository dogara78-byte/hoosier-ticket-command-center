# Patch 14 - Season Selector + Audit Trail

Adds a dashboard season selector and audit tables for Money, Seats, Parking, and Settlement.

## Version

v2026.06.25-patch14-season-audit

## What changed

- Season selector: Active season, each detected season, and All seasons.
- Seat audit shows seat-column credits/payments, allocated seat costs, net balance, and rows hit.
- Settlement audit shows member-column credits/payments, allocated costs, net settlement balance, and rows hit.
- Parking audit lists scoped parking rows.
- Money page shows scoped positive rows, scoped cost rows, and scoped net activity.
- Public snapshot mode continues to work; snapshot rows are audited using the same logic.
