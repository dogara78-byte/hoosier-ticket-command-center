# Patch 28F - Season Reset on Page Change

This patch resets the dashboard season selector to Active season whenever the user changes pages.

Member behavior:
- If a member selects 2024 or 2025 on one page, that choice does not carry into the next tab.
- Every page opens on Active season, currently 2026.
- The user can still change the season within that page for comparison.
