# Patch 17 - Member-First Dashboard Redesign

This patch moves the Score/Game Day Dashboard away from developer/admin signals and toward what Joel, Kyle, and other members care about.

## Main changes

- Score page now leads with 2026 fund status, member status, ticket sales, parking sales, next activity, and last activity.
- Workbook row count and latest transaction ID are moved into a small Data Status note instead of headline cards.
- Member fund cards emphasize everyone is settled for 2026 and show roll-forward/cash-paid context.
- Version updated to `v2026.06.25-patch17-member-dashboard`.
